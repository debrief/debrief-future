# Welcome to Debrief Loader Storybook

This Storybook showcases the UI components for the **Debrief Loader** - a desktop application for loading maritime data files into STAC catalog plots.

## Beta Preview

You are viewing a **beta preview** of the Loader UI components. We welcome your feedback to help us improve the user experience before the full release.

## Provide Feedback

Have suggestions, questions, or feedback about these components?

**[Visit our GitHub Discussion](https://github.com/debrief/debrief-future/discussions/categories/loader-feedback)**

Please share:
- Usability concerns
- Accessibility issues
- Visual design suggestions
- Workflow improvements

## Component Categories

### Core Workflow
- **StoreSelector** - Step 1: Select destination STAC store
- **PlotConfig** - Step 2: Configure plot (create new or add to existing)
- **ProgressView** - Processing status display
- **SuccessView** - Completion confirmation
- **ErrorView** - Error handling with actionable messages

### Supporting Components
- **StoreCard** - Individual store display
- **PlotCard** - Individual plot display

## Getting Started

1. Browse components in the sidebar
2. Interact with controls to see different states
3. Use the accessibility addon to check a11y compliance
4. Share feedback via the Discussion link above
