# LinkedIn Shipped Summary

Screenshots don't let readers interact with your components. Videos are passive. We extended our speckit workflow to bundle Storybook stories as self-contained JavaScript files and embed them directly in blog posts.

The planning phase identifies bundleable components. Implementation builds esbuild bundles with React and CSS inlined. Publishing deploys them alongside posts to GitHub Pages.

Result: Live, interactive component demos inline in every shipped post. Readers explore the real UI without leaving the article.

Five markdown files modified. Zero new dependencies. The next feature with visual components will automatically offer this capability.
