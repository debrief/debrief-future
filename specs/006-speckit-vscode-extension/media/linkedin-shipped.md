**Shipped: Debrief VS Code Extension**

Excited to announce the release of the Debrief VS Code Extension - maritime tactical analysis directly in your code editor.

**What it does:**
- Browse STAC catalogs from VS Code Explorer
- Display vessel tracks on interactive Leaflet maps
- Select tracks with intuitive click interactions
- Zoom, pan, and fit bounds with floating toolbar

**Technical highlights:**
- TypeScript with strict mode
- Comprehensive unit test coverage
- Canvas rendering for 10K+ points
- Webview state persistence

This is Stage 6 of our tracer bullet delivery - the UI layer that brings the entire Debrief platform together.

Try it: Search "Debrief Maritime Analysis" in VS Code Extensions.
